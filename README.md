# QR Code Generator

Este proyecto es una aplicación web que permite a los usuarios generar códigos QR de manera rápida y sencilla. Puedes visitar la aplicación en [QR Code Generator](https://nextjs-boilerplate-theta-ten-75.vercel.app/).

## Características

- **Entrada de URL o Texto**: Permite ingresar cualquier URL o texto para generar un código QR.
- **Generación de Código QR**: Genera un código QR basado en la URL o texto proporcionado.
- **Interfaz Amigable**: Fácil de navegar y usar.
- **Diseño Responsivo**: Compatible con dispositivos de escritorio y móviles.

## Tecnologías Utilizadas

- **Frontend**: Next.js, React
- **Estilos**: CSS, Tailwind CSS
- **Generación de QR**: Librería `qrcode`

## Instalación y Configuración

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tuusuario/qr-code-generator.git

2. Navega al directorio del proyecto:

cd qr-code-generator

3. Instala las dependencias:

npm install

4. Inicia el servidor de desarrollo:

npm run dev

5. Abre tu navegador y ve a http://localhost:3000 para usar la aplicación.
Uso- Abre la aplicación en tu navegador.
6. Ingresa una URL o texto en el campo de entrada.
7. Haz clic en el botón "Generar Código QR".
8. El código QR se mostrará en la página, listo para ser escaneado o descargado.
